Function msToTime

  • msToTime(milliseconds, format): number

  • Returns a time object (or a converted equivalent if a format is provided) converted from milliseconds.

    Parameters

    • milliseconds: number

      The milliseconds to convert.

    • format: TimeFormats

      The format to convert to. Accepts 's' for seconds, 'm' for minutes, 'h' for hours, 'd' for days.

    Returns number

    TimeObject or the converted equivalent if a format is provided.

    Reference

    Example

    msToTime(1000 * 60 * 60 * 24 * 2 + 1000 * 60 * 60 * 3 + 1000 * 60 * 4 + 1000 * 5 + 6)
